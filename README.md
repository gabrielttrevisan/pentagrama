# Rodando o Projeto

## Versões

-   PHP 8.1.10
-   Node 16.13.1

## Comandos

1. composer install
2. npm install
3. ./vendor/bin/sail up -d
4. docker-compose exec -it laravel.test bash
5. php artisan migrate:fresh --seed

Nota: se estiver em ambiente Windows, utilize o WSL.
